QuizziPedia

University Project

Description

QuizziPedia is a Software Web that allows the creation and management of questionnaires able to detect the gaps of candidates before,during and at the end of a training course. The system offers the following features:

  • Store questionnaires in a server divided by topic;
  • Give the user, via an interface, specific questionnaires chosen topic;
  • Check and evaluate the questionnaires chosen by users according to the answers given.
